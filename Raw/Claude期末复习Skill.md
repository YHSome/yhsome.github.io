# 这是什么

一个 Claude Code Skill，专为期末复习设计。你只需要把老师的课件（PPT、PDF、Word 等格式）丢给它，它就能自动识别内容、提取知识点、生成一套可视化的复习讲义。

# 为什么做这个

期末复习最痛苦的事不是背书——是**整理资料**。

老师发的 PPT 几十页、课本 PDF 几百页、自己记的笔记东一张西一张。考前想系统过一遍知识点，光是把这些资料通读一遍就要花半天。更别说还要梳理逻辑、画思维导图、做重点标记。

"能不能让 AI 帮我读课件、做讲义？"

Claude Code 支持自定义 Skill——可以给它设定角色、规则和工作流程。于是就有了这个项目：定义一个专门用来"备考"的 Skill，让它像助教一样帮你整理复习资料。

# 它能做什么

1. **多格式识别**：PPT、PDF、DOCX 统统扔进去，不用手动转格式
2. **知识点提取**：自动识别章节结构、关键概念、定义、公式、例题
3. **可视化讲义**：用 Mermaid 图表展示知识结构，包含思维导图、流程图、时间线
4. **重点标注**：根据内容密度和重复频率自动标记高频考点
5. **生成复习大纲**：输出一份结构化的 Markdown 讲义，可直接打印

# 使用方式

在 Claude Code 中安装 Skill 后：

```
/期末复习 上传课件/
  把高数第三章的PPT和课本PDF丢进去
```

Claude 会：

```
1. 扫描 PPT → 提取 12 个关键概念、8 个公式、6 道例题
2. 对照 PDF → 补充 5 个定理证明
3. 生成思维导图 (Mermaid)
4. 输出 Markdown 讲义 → 保存为 高数第三章-复习讲义.md
```

# 技术实现

基于 Claude Code 的 Skill 系统，定义了三层结构：

**第一层：文件解析**。利用 Claude 原生的文件理解能力，自动识别上传的 PPT/PDF/DOCX 内容。不需要额外的 OCR 或解析库——Claude 的多模态能力直接处理。

**第二层：知识提取**。通过精心设计的 prompt 模板，引导 Claude 按照"章节结构 → 核心概念 → 关键公式 → 典型例题"的层次递进提取。

**第三层：可视化输出**。用 Mermaid 语法生成图表——思维导图展示知识层级、流程图展示推导步骤、时间线展示历史事件。Claude 原生支持 Mermaid 渲染，生成的讲义可以直接在支持 Markdown 的编辑器中查看。

# 实际效果

以高中物理"电磁感应"一章为例：

**输入**：老师发的 35 页 PPT + 课本第 6-8 章 PDF（共 60 页）

**输出**：
- 核心概念卡片 × 18（磁通量、法拉第定律、楞次定律、自感互感…）
- 公式表 × 12（含适用条件）
- Mermaid 思维导图 × 1（一章知识全景）
- 典型例题 × 8（含解题思路）
- 易错点提醒 × 5（方向判定、单位换算…）

**生成时间**：约 3-5 分钟

# 与 ChatGPT 的区别

很多人问：这和直接把课件丢给 ChatGPT 有什么不同？

区别在于**专精**。通用的 ChatGPT 对话不会主动帮你做"期末复习"这件事——你需要反复调整 prompt、手动筛选输出、自己画图。而这个 Skill 把整个工作流编码成了一套固定的指令序列，每次执行都按同一个标准来——就像你雇了一个知道你喜欢什么风格、熟悉你学科内容的私人助教。
